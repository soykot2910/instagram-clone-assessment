#### Technologies Used
- React

#### Project installation
1. Clone Repository
```
$ git clone https://github.com/soykot2910/facebook-clone-assessment.git
$ cd facebook-clone-assessment
```
2. Install dependencies

```
$ npm i
```
3. Run Project
```
$ npm start
```
